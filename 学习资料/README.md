## 💡 学习技巧

1. **使用 IDE 的跳转功能**：在 VS Code 中，按住 Cmd/Ctrl 点击函数名可以跳转到定义
2. **断点调试**：在关键位置添加断点，观察执行流程
3. **画流程图**：用纸笔画出数据流、渲染流程、事件流
4. **修改测试**：尝试修改代码，观察效果变化
5. **阅读文档**：结合在线文档 https://www.klinecharts.com 理解 API 设计
